# The Release of UASIP Metric.
This is the code of the implementation of the Unified No-Reference Quality Assessment for Sonar Imaging and Processing (UASIP). 

# Requirment
Matlab.

# Quick Start
run 'demo.m'
(Before starting, please download MatLab tool package 'matlabPyrTools' from https://github.com/gregfreeman/matlabPyrTools.)
